# SmartHomeSwitches
I built a simple Box containing 4 Smart-Home Switches. The Brain of the Switches is a Esp8266-01 Wifi Module.

Included Libraries:
  - <Arduino.h>
  - <ESP8266WiFi.h>
  - <ESP8266WiFiMulti.h>
  - <WebSocketsClient.h>
  - <ArduinoJson.h>
  - <StreamString.h>
